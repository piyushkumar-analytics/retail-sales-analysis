# 📊 Retail Store Data Analysis using Python

## 📌 Project Overview
This project focuses on data cleaning and exploratory data analysis (EDA) on a retail store dataset to understand customer behavior and sales patterns.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Numpy
  
---

## 🔧 Data Cleaning
Performed in: `retail_store_data_cleaning.ipynb`

- Handled missing values using median
- Removed unnecessary columns
- Treated outliers
- Fixed data types (datetime, category, etc.)

Cleaned dataset saved as: `cleaned_retail_store_data.csv`

---

## 📊 Exploratory Data Analysis (EDA)
Performed in: `retail_store_eda.ipynb`

- Sales by Category
- Sales by Location (Online vs In-Store)
- Payment Method Analysis
- Monthly Sales Trend
- Price Distribution
- Quantity Distribution

---

## 🔍 Key Insights
- Sales are evenly distributed across categories
- Online and in-store sales contribute almost equally
- Payment methods show balanced usage
- Most products fall in the mid-price range
- Customers tend to purchase moderate quantities
- Sales remain stable across months

---

## 📌 Conclusion
The analysis shows that the business has stable and balanced performance across categories, locations, and time.  
Customer behavior is consistent, and there is no strong dependency on a single factor.

---

## 📂 Project Files
- `retail_store_data_cleaning.ipynb` → Data cleaning
- `retail_store_eda.ipynb` → Data analysis & visualization
- `retail_store_data.csv` → Raw dataset
- `cleaned_retail_store_data.csv` → Cleaned dataset
